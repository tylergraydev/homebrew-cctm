# Homebrew Tap for Claude Code Tool Manager

This is the official Homebrew tap for [Claude Code Tool Manager](https://github.com/tylergraydev/claude-code-tool-manager).

## Installation

```bash
brew tap tylergraydev/cctm
brew install --cask claude-code-tool-manager
```

## Update

```bash
brew upgrade --cask claude-code-tool-manager
```

## Uninstall

```bash
brew uninstall --cask claude-code-tool-manager
brew untap tylergraydev/cctm
```
